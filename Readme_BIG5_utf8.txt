自動防火牆/NAT 支援

紅色警戒二現在自動支援以上所列出之防火牆，不需進行任何困難的防火牆設定，諸如埠轉址、靜態鏡像或其
他複雜的設定。

支援功能為相當完整且容易使用，包含以下功能：

1. 支援當於防火牆之後，主持或加入任何遊戲。
2. 支援於同一防火牆之後的多人連線遊戲；假如您擁有一台以上的電腦區域網路，共用 Cable modem、DSL 
   或其他連線，您可以進行對抗或與其他網際網路上的玩家連線。

現在當遊戲於 Westwood Online 啟動時，您會看到 "連線至其他玩家" 的對話框，此處您可以看到與遊戲中
其他玩家的連線。假如玩家無法建立連線，遊戲將不會啟動，並將顯示訊息解釋誰遇到連線問題。

在有些防火牆產品，於嘗試與其他玩家建立連線前，必須等待接收其封包。假如您的防火牆使用此運作機制，
您必須於網路選項中開啟 "傳送延遲" 功能。於主選單，選擇 "選項" 然後選擇 "網路"。於不非必須狀況，
假如使用此設定可能會導致與其他玩家連線遭遇問題，請參考您的防火牆說明，以確定防火牆操作。（注意：
 Netgear 防火牆型號 RT311 & RT314 於測試必須使用此選項）

防火牆廠商請注意：假如您的防火牆產品並未列於已測試產品列表，我們很樂意傾聽您的意見！請聯絡 
Westwood Studios gameresearch@westwood.com。 


手動防火牆/NAT 支援

除了自動防火牆支援功能外，我們也包含了進階設定供進階玩家使用，假如其他方法都失效。於主選單中選擇
 "選項" 然後選擇 "網路"，於 "網際網路防火牆設定" 區域下，您會看到 "埠號碼" 選項，用以指示紅色警
 戒二使用指定的埠號碼。您必須指定介於
1024 與 65535 間的埠號，每一部要進行遊戲的電腦都必須給予不同的埠號碼，假如都於同一個防火牆之後。
欲回復到自動模式，移除 "埠號碼" 編輯盒內的埠數字即可。

假如您指定一組埠供紅色警戒二使用，您必須手動於防火牆設定一組靜態 IP 及埠轉址路由封包至正確的電腦
上。每一種防火牆設定方法不同，因此請參考您的防火牆文件以確定如何進行。您可能必須為每台電腦指定一
組靜態 IP 以連線至防火牆，否則於重新啟動電腦或防火牆後，靜態轉址規則將無法運作。


於 Windows 98 中使用兩片網路卡

Windows 98 作業系統一次只能擁有一個網路閘道，這表示如果您 Windows 98中執行紅色警戒二，並擁有兩張
以上的網路卡，您必須確認所有資料是透過正確的網路卡進行。最佳的方法就是停用非用以連線至網際網路/
區域網路的網路卡。否則您將會遭遇到紅色警戒二連線問題。

大部分音效卡問題都可以藉由更新音效卡最新版本驅動程式以獲得解決，請參閱您的音效卡手冊以瞭解如何更
新最新版本驅動程式的方法。

假如您使用 USB 喇叭，請不要遊戲進行中關閉喇叭。這樣可能會導致遊戲當機。

	2.3. 已知 DirectX 問題：

假如於其他程式正使用音效卡時進行安裝或執行遊戲，將會出現"DXSound Error"錯誤。假如發生此種情形，
請關閉所有程式，並在執行安裝或執行遊戲。

	2.4. Gateway電腦用戶：

假如您使用 Gateway 電腦並安裝 WildFile "Gateway GoBack" 軟體，於背景執行時，可能於紅色警戒二中遭
遇當機的問題。GoBack 載入於視窗系統上（於時間顯示旁）並顯示為兩個黃色箭頭。欲關閉GoBack，於工作
列黃色箭頭上點選右鍵並選擇關閉。建議您於執行紅色警戒二前關閉所有背景執行程式。

	2.5. 虛擬記憶體設定：

終極動員令：紅色警戒二可能需要遠較您系統高的記憶體，Windows 95/98 將自動以眾所皆知的"虛擬記憶體
"方式管理此工作 - 即使用您的硬碟空間以模擬需要記憶體、置換資料及調整您的硬碟。Windows 95/98 允許
您手動設定硬碟空間大小以作為虛擬記憶體：我們強烈的建議您不要執行此一動作！欲讓 Windows 95/98 管
理虛擬記憶體，請至Windows 95/98 開始、選擇'設定\控制台'，然後雙擊鍵'系統'並選擇'效能\檔案系統\虛
擬記憶體'，選擇'讓Windows管理我的虛擬記憶體設定值 - 建議使用'。

	2.6. 已知RealPlayer問題：

RealPlayer 似乎會與紅色警戒二影片播放軟體發生相衝，當RealPlayer之"Start Center"功能開啟（將保持
RealPlayer於常駐狀況），我們發現會發生遊戲不穩定狀況，會造成紅色警戒二如退出遊戲般，跳回視窗桌面
。

	2.7. 自動播放：

假如您的光碟於置入光碟機後無法自動播放，您可能是將此功能關閉了。欲開啟自動播放，右鍵點選'我的電
腦'並選擇'內容'然後選擇'裝置管理員'。選擇您的光碟機並點選'內容'，選擇'設定值'並點選'自動安插通知
'核選盒。

	2.8. 16位元光碟機驅動程式：

假如於AUTOEXEC.BAT或CONFIG.SYS中載入16位元光碟機驅動程式，可能會導致終極動員令：紅色警戒二無法啟
動。假如您使用MSCDEX或類似驅動程式，試著自autoexec.bat及config.sys移除它們。Windows 95/98 並不需
要此些驅動程式以驅動光碟機，並可以獲得比較好的效能。

	2.9. 光碟置換機及多台光碟機：

終極動員令：紅色警戒二並不支援多台光碟機或光碟置換機（即一台光碟機，可置入多片光碟片），並預設為
您只有一台光碟機。假如您擁有一台以上的光碟機，您必須於光碟機代號叫前面的光碟機中置入終極動員令：
紅色警戒二的光碟。例如：假如您擁有光碟機E：、F：及G：，您的終極動員令：紅色警戒二光碟必須置於E：
。

	2.10. 影片效能不佳：

假如您發現影片播放效能不佳或不穩定，試著縮小Windows 95/98 光碟快取記憶體大小，進入開始/設定/控制
台並雙擊鍵系統。於系統內選擇效能頁面，並點選檔案系統鍵。於檔案系統內容視窗，選擇CD-ROM 並注意快
取記憶體設定（假如必要時，您可以重新設定快取記憶體）。現在將快取記憶體設定為小。此動作應該有助於
改善影片效能。

您也能試著於遊戲選項選單調整畫面精細度。較低的畫面精細度將會移除如半透明效果及建築物動畫。紅色警
戒二也會於較低階電腦動態移除某些狀況動畫以確保高品質的遊戲效能。

	2.11. 省電模式：

由於電腦長時間沒有動作，是可能導致您的電腦與ISP斷線的，有時候是因為您的"省電"選項睡眠模式所造成
的，當遊戲進行或連線至Westwood Online也可能發生此問題，建議您於遊戲進行時，關閉您的螢幕保護程式
或省電模式。
	
	2.12. 週期性效能低落：

假如於執行終極動員令：紅色警戒二，卻不是於讀取光碟資料時，發生週期性的效能低落，請確認沒有其他背
景程式正在執行，這些程式包含防毒軟體。

	2.13. 遊戲更新與修正檔：
		
終極動員令：紅色警戒二包含自動更新功能。當您執行Westwood  Online時，將會自動通知更新版本的訊息。
當有更新版本可供下載時，您將會收到通知，假如願意的話，您可以立即自動下載並安裝。假如您並沒有連線
至Westwood Online 執行終極動員令：紅色警戒二，請參閱HTTP://www.westwood.com以獲得任何最新消息。

	2.14. 連線速度調整：

紅色警戒二於遊戲選項畫面中擁有遊戲潛在極限設定，允許玩家增加網路潛在極限以進行網際網路遊戲。欲調
整此項目，請調整至適當設定。例如：假如您使用數據機進行連線，則最佳的設定為"正常"或"差"。假如您採
用寬頻連線（Cable Modem或DSL），調整調整器至"快速"。此動作將嘗試調整您的系統達到效能極限。請不要
自作聰明的將其設定為"甚差"。正確的調整將有助於改善效能，而非減低效能。

	2.15. 使用兩張網路卡及 IPX 問題： 

假如您使用的電腦安裝兩張網路卡，並同時使用IPX通訊協定，您可以時"網路選項"以選擇您所要使用的網路
卡。注意：於某些Windows 95/Windows 98系統及某些設定，紅色警戒二將無法找到您的第二張網路卡，解決
此問題的方法為視窗裝置管理員關閉您的主要網路卡，並重新開機。紅色警戒二應該可以成功的找到並設定您
的第二張網路卡。請參考您的視窗相關文件以瞭解如何關閉您的主要網路卡。

	2.16.  防火牆問題：

假如您於防火牆之後執行紅色警戒二，並希望能夠主持網路遊戲，您必須開啟以下通訊埠：埠1234、1235、
1236、1237。此些埠使用UDP通訊協定。假如您不確定如何設定您的防火牆，請參考您的防火牆文件或聯絡您
的系統管理員或網際網路提供服務廠商。

	2.17.  網路轉址（NAT：Network Address Translation）遊戲：

假如您的網路使用網路轉址，位於防火牆之後的兩位或以上的玩家將無法加入同一場遊戲。但是他們可以參加
不同的遊戲。






3.	遊戲進行資訊與更新

	3.1. 工程師：

工程師（與搭乘 IFV 的工程師）能夠如其他部隊一樣，進入警戒模式。當於警戒模式時，工程師會自動修復
附近任何玩家所擁有的受損嚴重的建築物或佔領敵軍建築物。


	3.2.	於Westwood Online 積分賽及錦標賽中排名：

Westwood 為玩家提供與全世界最強的玩家對抗，測試其勇氣的機會。請參考Westwood.com以獲得最新消息。
紅色警戒中的積分賽及錦標賽排名預設至"快速遊戲"。

	3.3.	於Westwood Online 上的快速配對競賽：

紅色警戒二提供玩家於戰場上快速相遇的簡易方法，只要試試於Westwood Online 上的 "快速配對競賽" 選項
，此選項將試著依相似的等級、相近的網路速度 Ping 值（因ISP差異）、區域位置及系統規格進行配對。
